## Download

You can download the latest release of Roblox Dumper from the [GitHub Releases](https://github.com/nopjo/roblox-dumper/releases) page.

1. Go to the releases page
2. Download the latest ZIP file
3. Extract it to a folder of your choice

## Requirements

- Windows 10 or later
- A Roblox game which must be this [Roblox Game](https://www.roblox.com/games/6923982205/Dumper) or your own game which has the rblx file uploaded

## Files Included

After extracting, you should have:

- `roblox-dumper.exe` - The main dumper executable
- `cloudflare_tunnel.bat` - Script to start the Cloudflare tunnel

## Next Steps
